## Running the project locally

```bash
# Install dependencies
$ composer install

# Run migrations to create all database structure needed (Make sure that you have MySQL running on your machine)
$ php artisan migrate

# Run seeders to populate database with records
$ php artisan db:seed

# Run the application
$ php artisan serve

```
